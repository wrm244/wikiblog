这里推荐一款 Ubuntu 里简单好用的 GIF 录制软件：<HighLight>Peek</HighLight><br/>


**获取 Peek app 源**

```bash
sudo add-apt-repository ppa:peek-developers/stable
```

**更新源**

```bash
sudo apt-get update
```

**安装 Peek**

```bash
sudo apt-get install peek
```

使用方式很简单，打开 Peek，可以设置保存文件格式，调整窗口区域及大小即可点击左上角开始录制，或者在首选项里自定义快捷键
